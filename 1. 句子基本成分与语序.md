语句结构是语言表达的骨架。中文语句结构灵活，但核心逻辑清晰。我们可以从 **“基本成分与语序”**、**“结构类型”**、**“句式分类”** 和 **“复句关系”** 四个层面来系统理解。

### 1. 句子基本成分与语序

一个句子的核心通常由以下几种成分构成，其**基本语序是“主语 + 谓语 + 宾语”**，这与英语相似，但修饰语（定语、状语）的位置差异很大。

| 成分     | 功能                                    | 位置         | 示例                                  |
| :------- | :-------------------------------------- | :----------- | :------------------------------------ |
| **主语** | 陈述对象，句子主题                      | 句首         | **他** 在读书。                       |
| **谓语** | 陈述主语的动作或状态                    | 主语后       | 他 **在读书**。                       |
| **宾语** | 动作的承受者或关联对象                  | 谓语后       | 他在读 **书**。                       |
| **定语** | 修饰、限定名词（主语/宾语）             | **中心语前** | **(聪明的)** 学生读 **(厚重的)** 书。 |
| **状语** | 修饰动词/形容词（表时间、地点、方式等） | **中心语前** | 他 **[在图书馆] [认真地]** 读书。     |
| **补语** | 补充说明动作的结果、程度等              | **中心语后** | 他读书读得 **<u>很快</u>**。          |

**核心特点**：中文是“**主动宾**”语序，但**修饰语一律前置于中心语**（如“厚重的书”），这与英语将部分修饰语后置（如 “a book *of great weight*”）有根本不同。

### 2. 从结构上分类：单句、复句与特殊结构

- **单句**：由一个主谓结构构成，表达一个相对完整的意思。
  - 例：**太阳升起来了。** (主谓)
  - 例：**下雨了。** (无主句，也是单句)
- **复句**：由两个或以上意义相关的单句（分句）组成。下文的“复句关系”会详述。
- **特殊结构**：中文有一些标志性句型。
  - **“把”字句**：强调对宾语的处置或影响。结构：**主语 + 把 + 宾语 + 动词 + 其他成分**。
    - 例：**我把房间**打扫干净了。 (强调对“房间”完成了“打扫”这一处置)
  - **“被”字句**：表示被动意义，但不如英语被动语态常用。结构：**宾语 + 被 + (主语) + 动词 + 其他成分**。
    - 例：**杯子被(我)打碎**了。
  - **连动句**：一个主语发出连续动作。
    - 例：**我坐车去图书馆看书。**
  - **兼语句**：前一个动词的宾语，是后一个动词的主语。
    - 例：老师**请同学们**发言。 (“同学们” 是“请”的宾语，又是“发言”的主语)

### 3. 从语气/用途上分类：四大句式

- **陈述句**：叙述事实。如：北京是中国的首都。
- **疑问句**：提出问题。如：你喜欢中文吗？
- **祈使句**：表达请求或命令。如：请坐！ / 不要说话。
- **感叹句**：抒发强烈感情。如：这里的风景真美啊！

### 4. 复句的逻辑关系

复句通过关联词体现分句间的逻辑，这是表达复杂思想的关键。

| 关系     | 常见关联词（前/后分句） | 示例                                 |
| :------- | :---------------------- | :----------------------------------- |
| **并列** | 既…又…， 一边…一边…     | 他**既**聪明**又**勤奋。             |
| **承接** | 接着， 然后， 于是      | 他先发言，**然后**我开始补充。       |
| **递进** | 不但…而且…， 甚至       | 这房间**不但**宽敞，**而且**明亮。   |
| **选择** | 或者…或者…， 不是…就是… | **要么**你来找我，**要么**我去找你。 |
| **转折** | 虽然…但是…， 然而       | **虽然**价格贵，**但是**质量很好。   |
| **因果** | 因为…所以…， 既然…就…   | **因为**下雨，**所以**比赛取消了。   |
| **条件** | 只要…就…， 只有…才…     | **只要**努力，**就**能进步。         |
| **假设** | 如果…就…， 即使…也…     | **如果**明天下雨，我们**就**改期。   |
