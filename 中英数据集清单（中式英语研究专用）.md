## 4. 使用建议

- **中式英语研究**：中文词典 + 英文平行语料，用于分析词序、语法和翻译规律。
- **AI / NLP 模型训练**：可结合 CC-CEDICT、OPUS、Tatoeba 构建汉英平行训练集。
- **文案 / 翻译质量检查**：WordNet + THUOCL + CMU Pronouncing Dictionary 可以辅助生成对照规则。

## 使用建议

- **研究多语种翻译 / 中式英语生成**：优先使用 OPUS、Tatoeba、CCAligned 等平行语料库。
- **训练多语种语言模型**：推荐使用 mC4 或 OSCAR 数据集。
- **词汇分析 / 多语言查词**：Wiktionary、PanLex、Apertium 适合构建对照规则或字典工具。
